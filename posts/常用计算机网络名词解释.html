常用计算机网络名词解释

常用计算机网络名词解释

名词解释

计算机网络体系结构

  1. WAN：广域网（Wide Area Network）
  2. MAN：城域网（Metropolitan Area Network）
  3. LAN：局域网（Local Area Network）
  4. PAN：个人区域网（Personal Area Network）
  5. PDN：公用数据网络（Public Data Network）
  6. RFC：请求评论（Request For Comments）
  7. SAP：服务访问点（Service Access Point）
  8. PON：无源光网络（Passive Optical Network）
  9. EPON：以太网无源光网络（Ethernet PON）
  10. GPON：吉比特无源光网络（Gigabit PON）
  11. VPN：虚拟专用网络（Virtual Private Network）
  12. ISO：国际标准化组织（International Organization for Standardization）
  13. OSI/RM：开放系统互连基本参考模型（Open System Inter Connection Reference Mode）
  14. PDU：协议数据单元（Protocol Data Unit）

物理层

  1. STP：屏蔽双绞线（Shielded Twisted Pair）
  2. UTP：非屏蔽双绞线（Unshielded Twisted Pair）
  3. Hub：集线器
  4. ADSL：非对称数字用户线（Asymmetric Digital Subscriber Line）
  5. NIC：网卡（Network Interface Card）
  6. HFC：光纤同轴混合网（Hybrid Fiber Coax）

数据链路层

  1. CRC：循环冗余校验（Cyclic Redundancy Code）
  2. ARQ：自动重传请求（Automatic Repeat reQuest）
  3. FEC：前向纠错（Forward Error Correction）
  4. FCS：帧校验序列（Frame Check Sequence）
  5. ACK：确认帧（ACKnowledgement）
  6. MAC：介质访问控制（Medium Access Control）
  7. LLC：逻辑链路控制（Logical Link Control）
  8. FDM： 频分多路复用（Frequency Division Multiplexing）
  9. TDM：时分多路复用（Time Division Multiplexing）
  10. STDM：统计时分多路复用/异步时分多路复用（Statistic TDM）
  11. WDM：波分多路复用（Wavelength Division Multiplexing）
  12. CDM：码分多路复用（Code Division Multiplexing）
  13. CDMA：码分多址（Code Division Multiplex Access）
  14. ALOHA：ALOHA协议（Additive Link On-line HAwaii system）
  15. CSMA/CD：载波帧听多路访问/碰撞检测（Carrier Sense Multiple Access with Collision Detection）
  16. PPP：点对点协议（Point-to-Point Protocol）
  17. LCP：链路控制协议（Link Control Protocol）
  18. NCP：网络控制协议（Network Control Protocol）
  19. HDLC：高级数据链路控制（High-level Data Link Control）
  20. VLAN：虚拟局域网（Virtual LAN）
  21. FDDI：光纤分布数字接口（Fiber Distributed Data Interface）
  22. ATM：异步传递方式（Asynchronous Transfer Mode）
  23. RED：随机早期检测（Random Early Detection）

网络层

  1. IP：网际协议（Internet Protocol）
  2. IPsec：IP安全协议（IP security）
  3. TTL：生存时间（Time To Live）
  4. MTU：最大传送单元（Maximum Transfer Unit）
  5. NAT：网络地址转换（Network Address Translation）
  6. CIDR：无分类域间路由选择（Classless InterDomain Routing）
  7. ARP：地址解析协议（Address Resolution Protocol）
  8. RARP：逆地址解析协议（Reverse Address Resolution Protocol）
  9. DHCP：动态主机配置协议（Dynamic Host Configuration Protocol）
  10. ICMP：网际控制报文协议（Internet Control Message Protocol）
  11. AS：自治系统（Autonomous System）
  12. IGP：内部网关协议（Interior Gateway Protocol）
  13. EGP：外部网关协议（External Gateway Protocol）
  14. IGMP：因特网组管理协议（Internet Group Management Protocol）
  15. CGI：通用网关接口（Common Gateway Interface）
  16. MF：还有分片（More Fragment）

传输层

  1. UDP：用户数据报协议（User Datagram Protocol）
  2. TCP：传输控制协议（Transmission Control Protocol）
  3. MSS：最大报文段长度（Maximum Segment Size）
  4. RTT：往返时间（Round-Trip Time）
  5. RTO：超时重传时间（Retransmission Time-Out）
  6. OSPF：开放最短路径优先（Open Shortest Path First）
  7. AIMD：加法增大乘法减小（Additive Increase Multiplication Decrease）

应用层

  1. C/S：客户/服务器（Client/Server）
  2. DNS：域名系统（Domain Name System）
  3. FTP：文件传输协议（File Transfer Protocol）
  4. TFTP：简单文件传输协议（Trivial File Transfer Protocol）
  5. NFS：网络文件系统（Network File System）
  6. MIME：多用途网络邮件扩充（Multipurpose Internet Mail Extensions）
  7. SMTP：简单邮件传输协议（Simple Mail Transfer Protocol）
  8. POP：邮局协议（Post Office Protocol）
  9. IMAP：网际报文存取协议（Internet Message Access Protocol）
  10. UA：用户代理（User Agent）
  11. WWW：万维网（World Wide Web）
  12. HTTP：超文本传输协议（HyperText Transfer Protocol）
  13. HTTPS：超文本传输安全协议（HyperText Transfer Protocol Secure）
  14. URL：统一资源定位符（Uniform Resource Locator）
  15. HTML：超文本标记语言（HyperText Markup Language）
  16. SNMP：简单网络管理协议（Simple Network Management Protocol ）
  17. RIP：路由信息协议（Routing Information Protocol）
  18. BGP：边界网关协议（Border Gateway Protocol）
  19. NVT：网络虚拟终端（Network Virtual Terminal）

视频、音频与无线网络

  1. RTP：实时运输协议（Real-Time Transport Protocol）
  2. RTCP：实时运输控制协议（RTP Control Protocol）
  3. QoS：服务质量（Quality of Service）
  4. IntServ：综合服务（Integrate Service）
  5. DiffServ：区分服务（Difference Service）
  6. Wi-Fi：无线保真度（无线局域网同义词）（Wireless-Fidelity）
  7. BSS：基本服务集（Basic Service Set）
  8. ESS：扩展服务集（Extended Service Set）
  9. AP：接入点（Access Point）
  10. DS：主干分配系统（Distribution System）

信息安全

  1. PAP：密码认证协议（Password Authentication Protocol）
  2. DES：数据加密标准（Data Encryption Standard）
  3. Replay Atack：重放攻击
  4. ESP：封装安全有效载荷（Encapsulating Security Payload）
  5. AES：先进的加密标准（Advance Encryption Standard）
  6. DOS：拒绝服务（Denial of Service）
  7. PKI：公钥基础结构（Public Key Infrastructure）
  8. CHAP：口令握手鉴别协议（Challenge-Handshake Authentication Protocol）
  9. IDS：入侵检测（Intrusion Detection System）

其他

  1. API：应用编程接口（Application Programming Interface）
  2. ATU：接入端接单元（Access Termination Unit）
  3. EUI：扩展的唯一标识符（Extend Unique Identifier）
  4. FIFO：先进先出（First In First Out）
  5. IND：反向领域发现（Inverse Neighbor Discovery）
  6. MIB：管理信息库（Management Information Base）
  7. MRU：最大接收单元（Maximum Receive Unit）
  8. NAV：网络分配向量（Network Allocation Vector）
  9. ONU：光网络单元（Optical Network Unit）
  10. OUI：机构唯一标识符（Organizationally Unique Identifier）
  11. PCM：脉码调制（Pulse Code Modulation）
  12. RA：注册管理机构（Registration Authority）
  13. RFD：精简功能设备（Reduced-Function Device）
  14. AN：接入网（Access Network）
  15. IEEE：（美国）电气和电子工程师协会（Institute of Electrical and Electronic Engineering）
  16. IRTF：互联网研究部（Internet Research Task Force）
  17. MPLS：多协议标记交换（MultiProtocol Label Switching）
  18. SDH：同步数字系列（Synchronous Digital Hierarchy）
