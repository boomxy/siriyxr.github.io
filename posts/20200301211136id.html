重庆邮电大学2018年硕士研究生入学 《计算机网络（803）》真题答案解析

重邮2018计算机网络（803）真题

一、单选题

1. A
  各层的数据单元分别为：

物理层 数据链路层 网络层 传输层
比特流 帧 数据分组 报文段

2. D
  网络层的四个重要协议分别为：IP、ICMP、ARP、RARP。
  TCP、UDP属于传输层。

3. D
  URL是对可以从因特网上得到的资源的位置和访问方法的一种简介表示，相当于一个文件名在网络范围的扩展。
  一般形式：<协议>://<主机>:<端口>/<路径>

4. B
  IP地址分类：A类（1-126），B类（128-191），C类（192-223），D类（224-239）E类（240-255），127为保留地址。

5. C
  TCP建立连接为三次握手，断开连接为四次握手。
【17年单选17题类似】

6. B
  测试连通性主要使用Ping命令；Traceroute（UNIX中的名字，在Windows中为tracert），可以用来跟踪分组经过的路由。

7. D
  使用生成树算法的目的是为了避免兜圈子。从树没有环路的特性也可以推测出来。

8. D
  TCP/IP是我们主要学习的一个工业标准协议集，由于被广泛使用而成为事实上的国际标准。

9. B
  从IP地址可推测出其属于C类地址，网络号为24位，通过子网掩码最后8位与IP地址最后8位‘与’运算得子网号为120。

10. C
  物理层是为了在物理媒体之为上一层提供一个传输原始比特流的物理连接；网线属于传输介质不是物理层。

11. C
  IPv4地址为32位，IPv6地址为128位。本题没说是哪一种IP地址，但只有C选项可能是正确的。

12. B
  FTP文件传输协议。WWW提供网页浏览服务；Telnet提供远程连接服务；SMTP用于邮件服务器之间收发邮件和邮件客户代理向服务器发送邮件。

13. A
  CIDR无分类域间路由，由“网络部分+主机部分”组成，使用斜杠法表示。其目的是为了聚合子网。

14. C
  防火墙可以允许某些IP或禁止某些IP访问，即进行网络权限管理，其可以是纯软件形式的，不一定是硬件设备。

15. A
  ISP即Internet Service Provider，网络服务提供商，如电信、移动。

16. C
  URL一般形式：<协议>://<主机>:<端口>/<路径>

17. C
  C/S即Client/Server，客户/服务器。

18. B
  在分类目录搜索引擎中最著名的就是雅虎。国内著名的分类搜索引擎有雅虎中国、新浪、搜狐、网易等。

19. C
  全球最大最受欢迎的全文搜索引擎就是谷歌，另外两个著名的网站是美国微软的必应和中国的百度。

20. C
  超链接标签为：<a></a>;A选项是段落标签；B选项是字体标签。

  CSMA/CD工作方式为：先听后发，边听边发，冲突停发，随机重发。所以是一边发送一边检测冲突。

  RIP和OSPF都是找最短路径，只是度量标准不同。RIP是距离（跳数），而OSPF主要是费用、距离、时延、带宽等。一般提到最短距离更多的是指RIP。

24. F
  活动文档不等于动态文档，拥有视屏等资源的网页是活动文档，但不一定是动态文档，动态文档是指文档的内容是在浏览器访问万维网服务器时，才由应用程序动态创建。

三、名词解释

31. Wide Area Network；广域网

32. Open Shortest Path First；开放最短路径优先

33. Internet Protocol；网际协议

34. Dynamic Host Configuration Protocol；动态主机配置协议

35. Encapsulating Security Payload；封装安全有效载荷

四、简答题

36.
  从单个网络APPANET向互联网发展：TCP/IP协议的初步成型；
  建成三级结构的Internet：分为主干网、地区网和校园网；
  形成多层次ISP结构的Internet：ISP首次出现。
【教材课后习题1-5】

37.
  物理层的基本功能是在物理媒体上为数据端设备透明的传输原始比特流。

  1. 机械特性：指明接口所用的接线器的形状和尺寸、引线数目和排列、固定和锁定装置等等。
  2. 电气特性：指明在接口电缆的各条线上出现的电压的范围。
  3. 功能特性：指明某条线上出现的某一电平的电压表示何意。
  4. 过程特性：说明对于不同功能的各种可能事件的出现顺序。
    【教材课后习题2-5】

38.
  好处：不检验数据部分可以加快检验的过程，使转发分组更快。
  坏处：数据部分出现差错时不能及早发现。即使是到达了终点，目的主机中的IP也仍然不检查数据部分是否正确。当IP数据报的数据部分送交上面的运输层时，运输层的协议才检查收到的数据有无差错。
【教材课后习题4-11】

39.
  对实时话音数据的传输是不能使用TCP的。这是因为TCP传输话音数据时，只要一出现差错或丢失，TCP就要重传。这就产生了额外的时延，有时这种时延会达到很高的数值，使接收方无法容忍。
  实施话音数据传输使用UDP时，虽然有可能丢掉几个分组并不通知发送方重传，话音质量会差一点，但仍然能听懂，同时能保证实时性。因此只要应用程序接受这样的服务质量就可以使用UDP。
【教材课后习题5-15】

40.
  应用层协议需要DNS。
  运输层协议需要的是UDP（DNS使用）和TCP（HTTP使用）。
【教材课后习题6-10】

五、计算题

41.
  数据传输速率=$20000\times \log_2 16=80000$ b/s。
【15年简答6题类似】

42.
  计算过程如下：

  若数据的最后一位或两位变成了0，则在接收端CRC校验的时候都无法整除，因而能够发现错误。
  即使有CRC校验，也不能保证可靠传输，要实现可靠传输除了保证数据不出错，还要保证按顺序不重不漏地接收。

43.
（1）使用集线器属于共享式以太网，每个站点的带宽为100/10=10Mbps，同一时刻只能保证一个站点通信。

（2）使用交换机属于交换式以太网，每个站点的带宽都为10Mbps，独占带宽，互不影响，这是交换机最大的优势。
【17年单选19题类似】

44.
  将三个IP地址分别与路由记录的子网掩码进行‘与’运算得出子网号，再与对应目的网络的子网号比对可得：
  192.16.2.96下一跳为：R3（没有与之对应的目的网络，则使用默认路由）；
  192.16.244.33下一跳为：直接，R1接口1；
  192.16.2.250下一跳为：R2

45.
（1）
  由两个报文序号分别代表本报文第一个数据的编号，则第一个报文的数据为70到89，可得第一个报文的数据长度为89-70+1=20B。

（2）B主机在收到第二个报文后返回的确认号是180，说明90～179正确接收，A主机第二个报文数据长度为90B。

（3）当A主机第一个报文丢失，而第二个报文B主机正确接收，则B应该继续向A主机请求第一个报文，即确认号为70。
【14年计算与综合3题类似、教材课后习题 5-32】

六、综合题

46.
（1）B；A；C

（2）死锁

（3）B

  拥塞控制是让网络能够承受现有的网络负荷，是一个全局性的过程，涉及所有的主机、所有的路由器，以及与降低网络传输性能有关的所有因素。相反，流量控制往往是指点对点的通信量的控制，即接收端控制发送端，它所要做的是抑制发送端发送数据的速率，以便接收端来得及接收。

47.
  从上到下，总左到右依次为：
  FIN=1；ack=u+1；
  ACK=1；seq=u+1；ack=w+1。

  在断开连接的4个TCP报文中，FIN和ACK都为1。
  当客户端首先方面断开连接后停止发送数据，等待服务器发送完数据并发出断开连接的报文，在其此期间服务器ack的值始终为收到的客户上一个TCP报文的seq加1（u+1）。
  当服务器发送完数据并发送的关闭连接报文的时候，客户端发送一个确认报文，seq为自己发送的上一个seq加1（u+1），ack为收到的上一个服务器报文的seq加1（w+1）。
