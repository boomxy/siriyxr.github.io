重庆邮电大学2017年硕士研究生入学 《计算机网络（803）》真题答案解析

重邮2017计算机网络（803）真题

一、单选题

1. B
  10BASE-5及10BASE-T、100BASE-T、吉比特、10吉比特使用的媒体访问控制方法都是CSMA/CD；IEEE 802.11无线网使用的是CSMA/CA。
  D选项FDDI是光线分布数字接口，不是媒体访问控制方法。

2. C
  协议是“水平”的，服务是“垂直”的。在A第4层加上的在B中第4层被读出。

3. C
  10BASE-T的数据传输速率为10Mb/s，每个码元对应一个二进制数，由于曼切斯特每个码元由两个电平组成，所以由奈奎斯特定理公式$10M=\frac{B}{2}\log_2(2)$得B=20M

4. B
  1坚持是一直监听信道，一旦监听到信道空闲就立即发送报文；非减持在监听到信道繁忙后会随机等待一个时间；p坚持会以p的概率发送，1-p的概率等待。相对来说1坚持冲突概率最高。

信道状态 1-坚持 非坚持 p-坚持
空闲 立即发送数据 立即发送数据 以概率p发送数据， 以概率1-p推迟到下一个时隙
忙 继续坚持侦听 放弃侦听，等待一个随机的时间后再侦听 持续侦听，等待下一个时隙 在侦听

5. B
  物理层传输单位是比特；数据链路层传输单位是帧；网络层传输单位是分组；传输层传输单位是报文。

6. C
  WAN是广域网；MAN是城域网；LAN是局域网；PAN是个人区域网；PDN是公用数据网络。校园网属于局域网。

7. C
  由于不考虑热噪声，所以采用奈奎斯特定理进行计算，传送速率$=2\times 6 \log_2(16)=48M$。

8. A
  虽然OSI参考模型分了很多层，但最终都只有物理层是在实体信道上进行信号收发到操作，而高层实际上都是在逻辑信道上进行处理，没有接触传输介质实体。

9. C
  C类IP地址的网络号范围是在192.0.0～223.255.255之间。

10. A
  UDP属于传输层协议，其他三个均属于网络层协议。

11. D
  SNMP简单网络管理协议（Simple Network Management Protocol ），在传输层是基于UDP的。
  由于SNMP要管理网络上的多台主机，如果每一台都要进行一个TCP链接显然是不明智的，使用无连接的UDP更为方便。

12. A
  Base64编码把报文划分成1024个单元，每个单元3字节。每个单元被编码为4字节，所以共有4096个字节。如果把这些字节每80字节划分为一行，将需要52行，所以需要添加52个CR和52个LF。$4096+52\times 2=4200$。
【14年简答3题原题】

13. A
  数据传输速率在数值上每秒传输的比特数，也就是带宽。

14. D
  光纤通信的原理是光的全反射，所以不会受到电磁干扰或噪声的影响。

15. B
  100BASE-T、10BASE-T的最大长度都是100米，10BASE5是500米。

16. C
  万维网传输使用的协议是HTTP超文本传输协议。

17. C
  TCP连接的建立被称为“三次握手”，连接的断开被称为“四次握手”。其他三个选项均是TCP协议拥塞控制的内容。

18. B
  VLAN之间要进行通信要涉及到网络层的子网路由，必须使用三层设备，即路由器、三层交换机等。

19. C
  如果采用共享以太网，则每台计算机均分带宽；如果采用交换以太网，则每台计算机带宽与网络总带宽相同，独占带宽，这也是交换机最大的优势。
【16年单选3题类似】

20. B
  OSI/ISO参考模型中，网络层的主要功能是路由选择、拥塞控制与网络连接；传输层提供可靠的端到端服务，透明的传送报文；数据链路层在通信实体之间传送以帧为单位的数据；表示层提供数据格式转换、数据加密与解密、数据压缩与恢复的功能。
【16年单选7题原题】

二、多选题

21. AB
  网络的传输方式可分为点对点（一对一）和广播式（多对多）。

22. BD
  VLAN之间要进行通信要涉及到网络层的子网路由，必须使用三层设备，即路由器、三层交换机等。

23. BD
  集线器工作在物理层，B错误；网桥能隔离冲突域，但不能隔离广播域，D错误。

24. ABCD
  物理层技术特性有：机械特性、电器特性、功能特性、规程特性。

25. AB
  子网掩码中“1”对应网络部分，“0”对应主机部分，网络部分又由网络号和子网号组成。子网掩码用于计算出IP地址的子网号，可将网络进一步划分为多个规模相同的子网络。

三、填空题

26. 面向无连接服务

27. .html

28. 草案标准
  由于“草案标准”容易和“互联网草案”混淆，从2011年10月起取消了“草案标准”这个阶段[RFC 6410]

29. 主机号

30. 71%
  由题干“隔层均采用最下开销”可知，TCP首部和网络层首部均为20B，以太网数据链路层MAC帧封装信息为18B，所以142/(142+58)=71%。

四、判断题

31. F
  OSPF是一种链路状态路由协议。RIP是基于距离向量的路由协议。

32. F
  分组交换又分为数据报和虚电路两种，其中数据报是无连接，虚电路是面向连接的。

33. T
  局域网主要是广播式网络，仅有物理层和数据链路层两层。

34. F
  卫星通信由于在大气中传播信号，有被截获的可能。

35. T

36. F
  IP属于网络层协议。

37. T
  协议三要素是语义、语法和同步。这里的同步并不是狭义的同频，而是广义的在一定条件下应当发生什么事情，因此有时同步也称规则或时序。

38. F
  采用争用型介质控制方法则会有冲突的现象，即使是CSMA/CA也不是完全避免冲突。但若采用令牌环网则没有冲突，令牌环网的环仅是逻辑上的环，物理上也可以是总线星型。也可使用多路复用技术解决冲突问题。

39. T

40. T
  被动攻击主要是指截获，可在用户不知情的情况下获得用户数据信息。
  主动攻击分为：中断、篡改、伪造

41. F
  数据报是无连接的，每个分组报文走的路线不一定相同，到达顺序也不一定和发送顺序相同。

42. F
  WAN是广域网；MAN是城域网；LAN是局域网；PAN是个人区域网。通过首字母缩写对应的英文单词来进行记忆。

43. T

44. T

45. F
  高层协议管理的是传输层及以上，管理的应用会话，而传输特性取决于下三层。

六、计算与简答题

51.
  许多用户通过复用技术就可以共同使用一个共享信道来进行通信。虽然复用要付出一定代价，但如果复用的信道数量较大，总的来看，在经济上还是合算的。常用的复用技术有：频分多路复用、时分多路复用、码分多路复用、波分多路复用。
【教材课后习题 2-13】

52.
  网络上负荷较轻时，CSMA/CD协议很灵活，哪个站想发送就发送，而且发生碰撞的概率很小。如果使用时分复用TDM，效率就比较低。但网络负荷很重时，CSMA/CD协议引起的碰撞很多，重传经常发生，因而效率大大降低。但这时TDM效率就很高。

53.
  IP首部中的源地址也可能变成错误的，要求错误的源地址重传数据报是没有意义的。
  不使用CRC可减少路由器进行校验的时间。
【教材课后习题 4-12】

54.
  发送方 UDP 对应用程序交下来的报文，在添加首部后就向下交付 IP 层。UDP 对应用层交下来的报文，既不合并，也不拆分，而是保留这些报文的边界。接收方 UDP 对 IP 层交上来的 UDP 用户数据报，在去除首部后就原封不动地交付上层的应用进程，一次交付一个完整的报文。
  发送方TCP对应用程序交下来的报文数据块，视为无结构的字节流（无边界约束，可分拆/合并），但维持各字节顺序。
【教材课后习题 5-8】

55.
  解析IP地址需要的时间是RTT1+RTT2+…+RTTn
若采用HTTP/1.0:
  建立TCP连接和请求万维网文档需要4RTTw
  共用时间RTT1+RTT2+…+RTTn+4RTTw
若采用HTTP/1.1:
  建立TCP连接和请求万维网文档需要3RTTw
  共用时间RTT1+RTT2+…+RTTn+3RTTw

  由于本题网页中只有一张图片，所以HTTP/1.1采用流水线和非流水线都一样
【12年计算与综合5题类似，教材课后习题 6-15】

56.

  1. 连接建立：发件人的邮件送到发送方邮件服务器的邮件缓存以后，SMTP客户就每隔一定时间对邮件缓存扫描一次。如发现有邮件，就使用SMTP的熟知端口（25）与接收方的邮件服务器建立TCP连接。
  2. 邮件传送：邮件的传送从MAIL命令开始。MAIL命令后有发件人的地址。下面跟着一个或多个RCPT命令，取决于把同一个邮件发送给一个或多个联系人。再下面就是DATA命令，表示开始发送邮件内容了。
  3. 连接释放：邮件发送完毕后，SMTP客户端应发送QUIT命令。SMTP服务器如同意释放TCP连接，邮件传送全部过程结束。
    【教材课后习题 6-23】

  B、C属于192.168.75.144/28

  子网掩码都相同，计算过后直接比较网络部分是否相同。

（2）192.168.75.160

  网络地址为网络号+子网号

（3）192.168.75.19～192.168.75.30与192.168.75.17

  注意：A的地址已经被使用，要排除掉。全0和全1的主机号也不能使用。

  该主机IP地址最后八位为10100100，可见子网号与D、E相同，属于统一网段，可以收到其广播，将主机部分全部置1得出广播地址为10101111即175
