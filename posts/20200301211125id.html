重邮2015计算机网络（803）真题

一、单选题

1. A
  以太网的行业标准是IEEE802.3。

2. A
  吞吐能力与单位时间传送数据量有关，宽带表示数据传输速率。

3. C
  脉冲编码调制（PCM），北美的24路PCM（T1）和欧洲的30/32路PCM(E1)。我国采用后者标准，T1的速率是1.544Mbit/s,E1的速率是2.048Mbit/s。注意D选项的MBps指的是字节而不是比特。
【14年单选3题类似】

4. A
  常见的信道复用技术有：频分复用、时分复用、码分复用、波分复用。
【14年单选4题原题】

5. A
  VLAN仅是从逻辑上进行了网络划分，并不是一种新型的局域网模型，处于同一个虚拟局域网的主机可能位于不同网段，同一网段的主机也可能分别位于不同的虚拟局域网中。
【14年单选4题原题】

6. C
  FTP文件传输协议，SMTP简单邮件传输协议，TELNET远程连接协议都是属于应用层的协议。

7. B
  $全双工交换机端口总带宽=端口个数\times 单个端口带宽 \times 2$
  谢希仁和王道的说法都是“N对”，但很多教材都不赞同，清华大学计算机网络的老师和四级网络工程师真题说法都是“N个”。
【14年单选7题原题】

8. B
  由于是C类地址，其网络号有24位，则子网号为3位。所以可划分子网数:$2^3-2=2$
  （RFC 950规定，对分类的IPv4地址进行子网划分时，子网号不能为全0和全1，但随着CIDR的广泛使用，现在有的路由器也支持全1和全0的子网号。则在做题的时候，对于指明分类的IPv4子网号要减2，而如果是CIDR则不用考虑）
【14年单选8题类似】

9. C
  TCP/IP体系与OSI体系第三层对应的是网络层。
【14年单选9题原题】

10. C
  X.25协议是一种面向连接的虚电路分组交换数据链路层协议。
【14年单选10题原题】

11. C
  由于CSMA有最小帧长的限制，同时封装信息长度固定，若发送帧小于最小长度则填0补齐。因此应使数据帧携带的数据信息尽量长，以提高有效信息在数据帧中所占比例，提高信道利用率。
【14年单选11题原题】

12. B
  OSPF采用的是链路状态算法，RIP采用的是距离向量算法。
【14年单选12题原题】

13. A
  HDLC是数据链路层的协议，工作在数据链路层和物理层。
【14年单选13题原题】

14. A
  流量控制是通过限制发送端发送数据的速率，防止超出接收端的接受能力而造成数据丢失，以此来确保数据的完整性的方法。
【14年单选14题原题】

15. B
  端到端通信并且是保证TCP的可靠性，则应使用滑动窗口技术，对错误数据进行选择重传，确保数据分组完整有序的到达。C、D选项都只能保证其中一种。
【14年单选15题原题】

16. D
  127.0.0.0为环路自检地址，此地址表示任意主机本身，目的地址为环回地址的IP数据报永远不会出现在任何网络上，不会被路由器转发。
【14年单选16题类似】

17. A
  CIDR使用掩码将IP地址分为网络前缀和主机号，但不对网络进行分类，使用“斜线记法”的方法进行表示，实现了对路由的聚合。
【14年单选14题原题】

18. C
  第三代移动通信网络即3G，现有三个无线接口国际标准，即美国提出的CDMA2000（中国电信使用），欧洲提出的WCDMA（中国联通使用）和中国提出的TD-SCDMA（中国移动使用）。

19. B
  ATM网络采用的固定信元长度为53B。
【14年单选19题原题】

20. A
  ISO/OSI参考模型中，传输层的主要功能是提供端到端的可靠传输，透明的传送报文。所谓端到端即主机进程之间的连接，而点到点仅指主机之间的连接。网络层的主要功能是路由选择、拥塞控制与网络连接；；数据链路层在通信实体之间传送以帧为单位的数据；表示层提供数据格式转换、数据加密与解密、数据压缩与恢复的功能。
【14年单选20题原题】

二、多选题

1. BC
  TCP、UDP属于传输层协议；IP属于网络层协议；SIP会话发起协议属于应用层。
【14年单选23题类似】

2. ABC
  数据还未发送则不可能得到确认，D错误。已发送的数据在收到确认前可能存在超时丢失或被破坏的情况，所以可再次发送。
【14年单选24题类似】

3. BCF
  路由器想要Ping通自己则需要在接口打环。
  主机IP地址17-24位为00110000，路由器IP地址17-24位为00100000。由于主机掩码为20位，路由器掩码为19位，所以主机用自己的掩码来计算的时候和路由器网络号不同不在同一网段，而路由器计算则相同认为在同一网段。

4. ACD
  三种交换技术是指：电路交换技术、报文交换技术、分组交换技术，而分组交换技术又分虚电路交换技术和数据报。

5. ABCD
  B选项中两个地址都是合法的，并且IP地址和MAC地址没有关系。

四、简答题

1.
  ARP即地址解析协议，是根据IP地址获取物理地址的一个TCP/IP协议。负责将某个IP地址解析成对应的MAC地址。

2.

  1. 域名系统是因特网的一项核心服务，它作为可将域名和IP地址相互映射的一个分布式数据库，能够使人更方便的访问互联网，而不用去记住能够被机器直接读取的IP地址。
  2. 当前因特网采用的是层次型命名机制，将名字空间分成若干子空间，每个机构负责进一步划分，授权给下一级机构管理。这样一来，名字空间呈现一种树形结构，树上每一结点都有相应标号。通过层次化的名字结构，减轻了单个机构的管理工作量，提高了效率。

3.
  传统以太网是共享性局域网，待用CSMA/CD协议，最小帧长必须大于整个网络的最大时延位（最大时延内可传输的数据位）。如果帧长度的太小，就可能出现网络上同时有两个帧在传播，就会产生冲突而造成网络无法发送数据。如果数据帧太长，会出现现有的工作站长时间发送数据失败，而且可能超出接收端缓冲区大小，造成缓冲溢出。由于多方面限制，每个以太网都有最小为64B，最大为1518B的长度规定，小于或者大于着限制都可视为错误的数据帧，一般的以太网转发设备会丢弃这些数据帧。
【14年简答1题原题】

4.
  公钥加密过程：

  1. 网络中的每个端系统都产生一对可用于将它接收的报文进行加密和解密的密钥。
  2. 每个系统都通过把自己的加密密钥放进一个登记本或者文件来公布它，这就是公开密钥。另一个密钥则是自己私有的。
  3. 如果A想给B发送一个报文，A就用B的公开密钥加密这个报文。
  4. B收到这个报文后就用它的保密密钥解密报文。其他所有收到这个报文的人都无法解密，因为B才有B的私有密钥。

  对称密钥机制实现身份鉴别：
  对称密钥加密的原理是通过一个对称密钥对数据进行加密，（如A有用对称密钥对数据进行加密后将加密的数据和密钥一并传送给B方，B放采用同一个密钥对这个加密数据进行解密，因为双方使用一个密钥故称对称加密）

5.
  碰撞域也称冲突域，是指连接在同一导线上的所有工作站的集合，或者说是同一物理网段上所有的结点的集合或以太网上竞争同一带宽的结点集合。这个与代表了冲突在其中发生并传播的区域，这个区域可以被认为是共享的。在OSI模型中，碰撞域看作第一层的概念。
  广播域是接受同样广播消息的结点的集合。由于许多设备极易产生广播，所以如果不维护，就会消耗大量的带宽，降低网络的效率。广播域被认为是OSI模型中的第二层的概念。
  碰撞域和广播域相同的都是会在同一个网段内发生，不同的是碰撞域有交换机可以区分，广播域只能由路由器来划分。

6.
  数据传输速率=$10000\times \log_2 32=50000$ bps。
【14年简答5题原题】

六、计算与综合题

1.
（1）
  最大数据长度为1400字节，包括20字节的固定首部和数据字段，且数据字段长度为8的倍数，因此有效数据字段的长度为1400-20-4=1376字节。
  这个IP数据报长度为2420字节，减去20字节固定首部长度，数据字段长度为2400字节，应当划分为$\lceil 2400/1376 \rceil=2$片。

（2）
  第一片：数据字段长度1376，片偏移0，MF=1；
  第二片：数据字段长度1024，片偏移172，MF=0

  注意：片偏移的单位为8B，虽然每个分片减去20B固定首部长度剩余1380B可用于装载数据，但由于数据长度必须是8B的倍数，所以一个分片实际最多可装有效载荷1376B。MF=1表示后面还有数据，MF=0表示这是最后一个。
【12年计算与综合题2题类似】

2.
（1）
  0x0050 0x4000 0x0000 0x000A 0x0000 0x0000 0x5012 0x1000 0x0000 0x0000

  这道题本身有问题，只给了一个序号，我们推测这个序号是服务器端的因为第二问要用到，那么第二次确认号我们并不知道是多少，在此设确认号为0暂且进行计算。
  第二次是从服务器发往客户端的，所以源端口为0x0050（80），目的端口为0x4000，序号为0x0000 0x000A（10），偏移（0101）、保留（000000）、标志位（010010）为0x5012，窗口为0x1000
  二进制每四位对应一个六进制数。偏移代表的数据部分在报文段中的偏移，其值就等于首部长度，但以4B位单位。

（2）
  11，1471

  序号字段值指的是本报文段所发送的数据的第一个字节的序号。起始序号10在TCP连接时消耗，所以第一个数据序号为11，第二个为11+1460=1471，两次即可传送完毕。

（3）
  滑动窗口的值为0，此时有可能是接收方的缓冲区满。

（4）
  当RST为1时，表明TCP连接中出现严重差错，必须释放连接。

3.
（1）
  子网1到6的子网掩码均为255.255.224.0

（2）
  R2:E2$\to$R2:E3$\to$R1:E1$\to$R1:E2$\to$R3:E3$\to$R3:E2$\to$R5:E3$\to$R5:E2
